# microservice-auth

This is Simple Auth Microservice using GRPC in different languages.

# Built with

- [GRPC](https://grpc.io/)
- [MongoDB](https://www.mongodb.com/)

## Gateway

- [Golang](https://go.dev/)

## Services

- [Nest](https://github.com/nestjs/nest)
- [TypeScript](https://www.typescriptlang.org/)
- [Python](https://www.python.org/)

# How to run

## golang-gateway

```bash
# change directory
$ cd golang-gateway

# run the app
$ go run main.go
```

## nest-auth
```bash
# change directory
$ cd golang-gateway

# run the app
$ npm start
```

## python-user
```bash
# change directory
$ cd golang-gateway

# run the app
$ 	python user_server.py
```
